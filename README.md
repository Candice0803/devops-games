# devops-games

## Déscription 
Ce projet est un jeu Javascript "Keep white space".
Le jouer (triangle) doit tirer afin d'éviter les ennemis.

## Installation
```bash
npm install

## Comment y-jouer ?
- Appuyer sur SPACE pour démarrer le jeu
- Utiliser les flèches pour tirer et abbtre les énnemis 
- Eviter les énnemis
- Survivre le plus longtemps possible
